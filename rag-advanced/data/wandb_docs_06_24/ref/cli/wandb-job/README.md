# wandb job

**Usage**

`wandb job [OPTIONS] COMMAND [ARGS]...`

**Summary**

Commands for managing and viewing W&B jobs

**Options**

| **Option** | **Description** |
| :--- | :--- |

**Commands**

| **Command** | **Description** |
| :--- | :--- |
| create | Create a job from a source, without a wandb run. |
| describe | Describe a launch job. |
| list | List jobs in a project |
